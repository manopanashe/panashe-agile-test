# Paloma Poster

Landing page built with React and Tailwind CSS

## Technologies Used
- HTML5
- CSS3 (Tailwind CSS)
- JavaScript (React)
- Babel

## Installation
1. Clone the repository:

```shell
git clone https://github.com/manopanashe/panashe-agile-test
```

2.Change directory to the project folder:
```shell
cd panashe-agile-test
```
